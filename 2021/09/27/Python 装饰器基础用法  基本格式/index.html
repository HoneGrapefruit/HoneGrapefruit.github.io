Python 装饰器基础用法 基本格式

装饰器：
用一个函数去装饰另一个函数或类，为其提供额外的能力
实现了一种名为代理模式的经典设计模式

​ 横切关注功能(cross-concern):跟正常的业务没有必然联系的功能
​ 这样的功能最适合用装饰器(代理模式)来实现
