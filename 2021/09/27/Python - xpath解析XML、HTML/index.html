加谓语(加条件)

语法:选中标签的路径[谓语]

1)、[N] — 获取同层的第N个标签

2)、[last()] — 获取同层的最后一个标签

[last() - N] — 获取同层的倒数第(N+1)个标签

3)、[position() > N]

[position() >= N]

4)、[@属性名] — 获取有指定属性的标签

[@属性名 = 属性值] — 获取指定属性为指定值的标签

5)、[标签 >/</>=/<=/= 数据] — 将标签按照指定子标签的内容进行筛选
